# SiistiArki verkkosivusto

Tämä on SiistiArki yrityksen virallinen verkkosivusto. Projekti tarjoaa siivous-, kodinhoito- ja pihanhoitopalveluita.

## Ominaisuudet
- Palveluiden esittely (siivous, kodinhoito, pihanhoito)
- Hintalaskuri (huom: vain arvio, ei lopullinen hinta)
- Yhteystiedot (sähköposti ja Instagram)
- Kevyt ja nopea staattinen sivusto

## Verkkosivusto
https://codod123.github.io/SiistiArki/

## Tärkeää
- Hintalaskuri antaa vain suuntaa antavan arvion
- Osa sisällöstä voi olla vielä keskeneräistä/väliaikaista
